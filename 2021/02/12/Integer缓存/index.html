Integer 缓存

2021-02-12 随笔 Comments

Integer 缓存是 Java 5 中引入的一个有助于节省内存、提高性能的特性。

IntegerCache的缓存机制： Integer是对小数据[-128,127]是有缓存的，再jvm初始化的时候，数据-128127之间的数字便被缓存到了本地内存中，如果初始化-128127之间的数字，会直接从内存中取出，不需要新建一个对象。

顺带：java在执行Integer a = -129 的时候进行了自动装箱，实际上会转变为Integer a = Integer.valueOf(-129)
